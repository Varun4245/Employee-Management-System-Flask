# Employee Management System

A full-stack Employee Management System developed using **Python Flask**, **MySQL**, **HTML**, **CSS**, **Bootstrap**, and **JavaScript**.

## Features

- Admin Login
- Dashboard Analytics
- Employee CRUD Operations
- Employee Profile
- Photo Upload
- Employee Search
- Department Filter
- Dashboard Charts
- Export to Excel
- Export to PDF
- Change Password
- Session Authentication
- Logout System

## Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Chart.js
- OpenPyXL
- ReportLab
